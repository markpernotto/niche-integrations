# Competition Registration Guide

## Steps to Register

1. **Visit the Competition Page**
   - Go to https://app.nicheandleads.com/competition
   - Review the competition details, prizes, and timeline

2. **Submit Registration**
   - Click "REGISTER NOW"
   - Fill out the registration form:
     - Full Name
     - Email
     - Company (optional)
     - GitHub URL (optional)
     - Target Integrations (select the ones you plan to build)
     - Introduction (optional)

3. **Get API Credentials**
   - After registration, you'll receive API credentials:
     - `CLIENT_ID` - OAuth client ID
     - `CLIENT_SECRET` - OAuth client secret
   - Add these to your `.env` file (see `.env.example`)

4. **Join Discord**
   - Join the competition Discord server for support and updates
   - Link will be provided after registration

5. **Start Building**
   - Use the integrations in this monorepo as a starting point
   - Each integration has its own README with setup instructions

## Competition Timeline

- **Deadline**: March 30, 2026
- **Prize**: $1,000 per qualifying integration + bonus prizes
